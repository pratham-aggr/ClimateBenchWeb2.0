About ClimateBench2

ClimateBench2 is a comprehensive framework designed to accelerate machine learning research in climate science. Building on previous benchmarks, ClimateBench2 offers enhanced datasets, more rigorous evaluation protocols, and greater accessibility.

Our mission is to foster collaboration between the climate science and machine learning communities by providing standardized tools that enable fair comparison of models and methodologies.

ClimateBench2 supports various climate modeling tasks including:

  • Weather forecasting at multiple time scales
  • Climate projections under different scenarios
  • Extreme event prediction
  • Climate downscaling and super-resolution

Available Datasets

ClimateBench2 provides access to the following datasets:

  • ERA5 Reanalysis: High-resolution global atmospheric reanalysis data from 1979 to present
  • CMIP6 Model Outputs: Climate model projections from the Coupled Model Intercomparison Project Phase 6
  • Satellite Observations: Remote sensing data from multiple satellite missions
  • Extreme Events Database: Curated collection of extreme weather events with associated meteorological data

All datasets are available in standardized formats with consistent spatial and temporal resolutions, making them ready for direct application in machine learning pipelines.

Benchmark Tasks

ClimateBench2 defines the following benchmark tasks to evaluate model performance:

  • Short-term Weather Forecasting: Predict weather variables 1-10 days ahead
  • Seasonal Forecasting: Predict climate anomalies 1-6 months ahead
  • Climate Downscaling: Generate high-resolution climate data from coarse inputs
  • Extreme Event Prediction: Forecast the occurrence and intensity of extreme weather events
  • Climate Sensitivity Assessment: Predict climate response to forcing scenarios

Each benchmark task includes standardized input/output specifications, training protocols, and evaluation metrics.

Evaluation Metrics

Models are evaluated using a diverse set of metrics:

  • Root Mean Square Error (RMSE): Measures average prediction error
  • Anomaly Correlation Coefficient (ACC): Quantifies pattern similarity between predictions and observations
  • Continuous Ranked Probability Score (CRPS): Evaluates probabilistic forecasts
  • Extreme Event Scores: Specialized metrics for extreme events (F1-score, Critical Success Index)
  • Physical Consistency Metrics: Measures compliance with physical constraints

Results are evaluated at multiple spatial scales (global, regional, local) and for different climate variables to provide comprehensive assessment.
